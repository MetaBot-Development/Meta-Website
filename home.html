    Features

    Mod Grief

    Make sure none of your server Staff members abuse the power and grief you server. Mod Grief will monitor all staff members and make sure your server stays safe, only interfiering when a rule has been triggered.

    Anti Alt

    Get notified when a possible Alt Account has joined your server. Meta will check every new account that joins and runs different scans to indentify wether the account is an Alt Account or not.

    Safe Bot

    Make sure only Admins can add bots to your server. Meta wont allow any other staff without Admin permissions to add bots to the server. This is to prevent malicious bots joining your server.

    Auto Mod

    Protect your server with simple Auto Moderation system that covers must be rules such as Racism, Self Botting & Discord invites. This is to keep your server clean but free for all the members!
