AutoMod

Discord AutoMod integration powered by Meta — easier, cleaner, and faster moderation setup that works even when Meta is offline.

Powered by Discord AutoMod

Meta no longer has a custom AutoMod system. Instead, Meta works directly with Discord's built-in AutoMod to provide you the best and most reliable moderation experience. By setting up AutoMod through Meta, you're configuring Discord's native AutoMod — but faster, cleaner, and with fewer steps. This system works even when Meta is offline or in maintenance.

⚠️
Permission Warning

Users with the Administrator permission will be ignored by AutoMod. Administrators can change the configurations of the module.

📋
Action Logs

All AutoMod logs will be sent into the server mod-logs channel. You cannot change the logging channel for AutoMod.

Why Use Meta for AutoMod?

Faster Setup

Configure all filters in seconds through Meta's streamlined interface instead of navigating Discord's settings.

Always Active

Because it uses Discord's native AutoMod, protection stays active even when Meta is offline or under maintenance.

Full Control

Modify any AutoMod rule anytime through Discord's server settings after the initial Meta setup.

Available Filters

Meta sets up the following Discord AutoMod filters for your server. Each filter targets a specific type of unwanted content.

Invite Filter
Active

Blocks users from sending Discord server invite links in chat. Prevents advertising and member poaching.

Racism Filter
Active

Blocks all well-known racial slurs and hate speech. Keeps your community safe and inclusive.

NSFW Filter
Active

Blocks all types of NSFW content and explicit material from being shared in your server channels.

Scam Filter
Active

Blocks all known Steam scam links, IP grabbers, and other malicious URLs from being shared.

Profile Filter
Meta Plus

Scans and filters out offensive user profiles, including inappropriate usernames and display names.

How to Modify AutoMod

After Meta sets up AutoMod, you can fine-tune any rule through Discord's server settings.

Editing AutoMod Rules

Navigate to your server settings to modify

To change how AutoMod behaves, head to your Discord server settings and find the AutoMod section:

Casual Server Settings → AutoMod
Community Server Settings → Safety Setup → AutoMod

From there, select the rule you want to edit and make your changes. All modifications take effect immediately.

How It Works

1

Run Setup

Use Meta's /settings command to enable and configure AutoMod filters for your server.

2

Discord Takes Over

Meta configures Discord's native AutoMod. The rules are enforced by Discord itself, not Meta.

3

Always Protected

Because Discord enforces the rules, your server stays protected 24/7 — even during Meta downtime.

💡
Pro Tip

You can add custom words and phrases to any AutoMod filter by editing the rules in Discord's server settings after Meta sets them up.
