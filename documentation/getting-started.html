Getting Started

Learn the basics of Meta and how to set it up for your Discord server. This guide will walk you through the initial configuration and get you protected in minutes.

📋
Prerequisites

Make sure you have Administrator or Manage Server permissions on the server where you want to add Meta.

Step 1: Invite Meta to Your Server

First, you'll need to add Meta to your Discord server. Click the button below or use our invite link to get started.

Add Meta to Discord

Click to authorize Meta with the required permissions

Invite Meta

Step 2: Run the Setup Command

Once Meta is in your server, run the setup command to configure basic settings. This wizard will guide you through the essential configurations.

Setup Command 
/setup

The setup wizard will help you configure:

  • Administrator Logs - Where Meta sends sensitive information, logs and messages.
  • Moderation Logs - Channel that will receive all mod-actions, AutoMod and Mod Command logs.
  • Moderator Rank - Role that will be monitored by Meta, to restrict certain actions.

Step 3: Configure Protection Modules

Meta comes with several protection modules that can be individually configured. Get started with protectiong the server your own way:

Protection Commands 
/settings

Configuration Options

Anti Mod Grief

Prevents compromised moderators from damaging your server.

Anti Mod Grief

Anti Alt

Identifies, blocks and automatically kick suspected alternate accounts.

Anti Alt

Malicious App Detection

Automatically detect, get notified and manage Malicious Applications and bots.

Malicious App

Auto Moderation

Automatically moderate messages, reactions, and user actions in your server.

Automod

Server Logs

Have a good overview of what's going on in your server with detailed and stylish logs.

Server Logs

Themes

Customize the look and feel of your server's Meta interface.

Appearance
⚠️
Important Note

Meta requires specific permissions to function properly. Make sure Meta's role is positioned above the roles you want it to manage in your server's role hierarchy.
