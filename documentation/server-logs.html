Server Logs

Comprehensive logging system that tracks every important event in your server. Stay informed about what's happening at all times.

How to Enable Logs

Each log category can be toggled individually
1
Run /settings in your server to open the configuration panel.
2
Navigate to the Server Logs section in the settings menu.
3
Click the corresponding button for each log type to enable or disable it.
4
Logs will be sent to your configured mod-log channel automatically.

Channel Filtering

Message Logs support channel filtering, allowing you to exclude specific channels from being logged. This is useful for bot channels or high-traffic channels you don't need to monitor.

How to Filter Channels

To add or remove channels from the filter, click the corresponding button in the Message Logs settings and submit the channel ID. The same process applies for both adding and removing filtered channels. To see all of the filtered channels, click on the Check button.

Add → Submit Channel ID
Check → Submit
Remove → Submit Channel ID
💡
Pro Tip

Only Message Logs support channel filtering. Other log types will capture all relevant events server-wide without the ability to filter by channel.
