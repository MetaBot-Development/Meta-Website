    Learn about all the commands and meaning of them!

    Select Commands Category

    Usage Examples:

    /scan

    This will quickly scan your server and provide you with some information.
    You can learn about the command fields in the Documentation.

    Usage Examples:

    /inspect @user

    This command will scan the user and provide general stats about the user such as join dates, roles, etc...
    For more information, you can learn about the command specifically in the Documentation.

    Usage Examples:

    /help

    Get the usual useful links and information about Meta bot.

    Usage Examples:

    /settings

    See all your server settings, what is enabled & disabled, what has been setted up and more.

    Usage Examples:

    /ping

    See Meta's ping, that's it....
    There is a 1% chance to get a special response, if you get it, save it and send it to the developers!

    Usage Examples:

    /setup [log-channel: #channel]
    /setup [mod-rank: @rank]
    /setup [meta-ban: create]
    /setup [protection: module]

    Set the bot up quickly with this 1 simple command. If you need help, seek support in the Support Server.
    For more information about the setup command, check out our Documentations.
