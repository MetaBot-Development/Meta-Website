Commands

Complete reference of all Meta Bot commands. Use slash commands (/) to interact with Meta in your Discord server.

Configuration Protection Moderation Info

Configuration Commands

/setup Essential

Initialize Meta in your server with a guided setup wizard. Configures basic protection settings.

Usage: /setup [admin-log] [mod-log] [mod-rank]
admin-log required Channel where sensitive information will be sent
mod-log required Channel where moderation logs will be sent
mod-rank required Role that will be monitored by Meta.
/settings Essential

The main configuration command for Meta. Change, set up and modify every single setting in Meta with the settings command.

Usage: /settings

Protection Commands

/inspect

Get detailed information about a specific user in the server. Inspect a user's profile, activity, status, ranks, permissions and much more.

Usage: /inspect [user]
user required The user you'd like to inspect
/scan

Advanced scanning system all in one command. Get detailed overview of your server instantly, detect issues, see stats and get an overall protection score.

Usage: /scan

Moderation Commands

/mute

Mute a user in the server with reason, duration and image of the wrongdoing.

Usage: /mute [user] [time] [time-format] [reason] [image]
user required The user you'd like to mute / time out
time required For how long you'd like to mute the user [NUMBER ONLY]
time-format required Choose the format [seconds, minutes, hours, days]
reason required What is the reasoning behind the mute
image optional Include a picture into the log for proof or just for fun :)
/unmute

Unmute a user in the server quickly with a solid reason option.

Usage: /unmute [ID] [reason]
ID required The ID of the user you'd like to unmute
reason required Specify the reason for the unmute
/ban

Ban a user from the server with a reason and optional image proof.

Usage: /ban [member] [reason] [image]
member required The user you'd like to ban
reason required Specify the reason for the ban
image optional Include a picture into the log for proof.
/unban

Unban a user from the server with a reason.

Usage: /unban [user] [reason]
ID required The ID of the user you'd like to unban
reason required Specify the reason for the unban
/kick

Kick a user from the server with a reason and optional image for proof.

Usage: /kick [user] [reason] [image]
member required The user you'd like to kick
reason required Specify the reason for the kick
image optional Include a picture into the log for proof.
