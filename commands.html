Commands

Slash Commands

Meta is equipped with Discord Slash Commands. All of the commands are usuable with the /command format. Some special / secret / easter-egg commands could be with prefixes.


Server Commands

/setup Administrator

Set up Meta is your server. This single command is all you need to run, to make your server safer!

Without setup the bot will not function in your server!


Inputs:

admin-logs - The channel where all of the sensitive logs will be sent.

mod-logs - Channel where all type of Moderation logs will be sent.

mod-rank - The rank that Meta will monitor and take action on if user has the rank.
/settings Administrator

Get a quick overview of your server Meta Settings. This command lets you manage all the protection systems and configurations.
Click on any of the buttons and start protecting your server like a true administrator!


Buttons:

Anti Mod Grief - Configurate the Anti Mod Grief Protection System.

Anti Alt - Modify the Meta Anti Alt Protection System. Make sure there are no fake accounts in your server.

Malicious Bot - Enable the advanced Malicious Bot Scanner to protect your server.

Automod - Choose the setup of your Discord AutoMod rules. Enable different rules and create them with 1 single click.
/scan Administrator

Scan your server and get a detailed overview of the channels, roles, administrators, alt accounts and much more.

Meta will also check your server for flaws and gives an overall protection score from 1 - 100 scale. To perfect your score, Meta gives you a list of issues on how you can improve the score.

Learn more about scanning here...


Buttons:

New Accounts - Get a summary of all accounts that are considered to be alt accounts.

Dangerous Roles - List of all the roles that are considered to be Dangerous Roles.

Admins - Get a list of all the users who have Administrator permissions in the server.

Bots - See every Discord Bot / App that is in your server.
/inspect Everyone

Get a detailed overview on someones Discord profile. From their activity to server roles, this command shows you all of the stats.


Input:

user - Choose an user whose profile you'd like to inspect.

Moderation Commands

/ban Moderator

Ban someone from the server and get all of the details instantly. Keep it clean, organized and professional.

Ban logs will be sent into mod-logs channel.
Banned user will be notified about the ban and the reason of the ban.
Moderator who went through with the ban will get insta overview as an "only you can see" message.


Inputs:

user - Choose the user who you'd like to ban from the server.

reason - Give a reason for the ban. Anything decent fits ;)

proof optional - Add an image of proof for others to see. Proof is included in log message if given.
/unban Moderator

Revoke a ban from a user whose already banned. Keep your server up-to-date and beautiful with amazing design.

Unban logs will be sent into mod-logs channel.
Unbanned user will not be notified about the unban.
Moderator who went through with the unban will get insta overview as an "only you can see" message.


Inputs:

id - Discord ID of the user you'd like to unban from the server.

reason - Give the unban it's very own reason. Be creative!
/mute Moderator

Time someone out in your server with clean and professional responses and actions.
Make people unable to chat and join voice channels in your server. Meta uses Discords Timeout feature to mute users.

Mute logs will be sent into mod-logs channel.
Muted user will be notified about the mute.
Moderator who went through with the mute will get insta overview as an "only you can see" message.


Inputs:

user - Choose the user you'd like to mute.

time - For how long you'd like to mute the user. Number Only

time-unit - Choose the unit. Seconds, minutes, hours or days.

reason - Give the mute it's reason, cannot mute for no reason right?

proof optional - Add a image as proof for others to see. Proof is included in the log if given.
/unmute Moderator

Make removing a timeout easy, clean and professional. Remove a timeout from an user with 1 single command, fast and easy.

Unmute logs will be sent into mod-logs channel.
Umuted user will be notified about the unmute.
Moderator who went through with the unmute will get insta overview as an "only you can see" message.


Inputs:

user - The user who you'd like to unmute.

reason - Give the unmute action a reason. Improvising is allowed!
/kick Moderator

Don't want to ban them but mute just doesn't do it? Kick the user from your server while staying professional and organized.


Inputs:

user - The user who shall be kicked from the server.

reason - Give the kick action a cool and reasonable reason.

proof optional - Add a image as proof for others to see. Proof is included in the log if given.

Casual Commands

/ping Everyone

Whenever the bot is online but is responding slowly, it's best to check if the issue is on Metas side. Check the apps ping with this command!

There is a 1% chance to get a special, super duper rare Mythic response when running this command. If you manage to be the one lucky fella, take a screenshot and send it to the Meta Team over at the Support Server. You'll receive a cool little suprize / reward for it.

Getting started with Meta? Don't worry, we have everything ready for you to start protecting your server! Check out the help command to get the most forward and easiet help directions.
