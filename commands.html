    • Meta Commands

    Learn about all the commands and meaning of them!


    /help



    Description:

    The basic information command that provides members with the links for Meta commands list, support server and the documentations.


    Permissions:

    This command is available for everyone. Only the command runner can see the response.





    /setup



    Values
    [ admin-logs: #channel ] - Where all the protection / sensitive logs will be sent.
    [ mod-logs: #channel ] - Where all the bans / unbans, timeoutes / timeout removals will be sent.
    [ mod-rank: @role ] - The rank that Meta will watch / monitor.



    Description:

    The quick setup command to get everything working! This is required, otherwise the bot cannot work or function in your server.
    With that command, you're setting the admin log channel, the moderation log channel and the moderator rank.


    Permissions:

    Only the people with Administrator permissions can run this command.





    /settings



    Description:

    The overview of your server. See every single setting and how Meta is setted up in your server. If the bot is setted up. This is the command where you can also modify the protection modules, by using the buttons below.


    Permissions:

    Only the people with Administrator permissions can run this command.





    /docs



    Values
    [ search: Document ] - What kind of document are you looking for.



    Description:

    Get a direct link to specific Meta Documentation document. Each section also includes a very short summary of the topic. But documents is more detailed version of them!


    Permissions:

    This command is available for everyone. Only the command runner can see the response.





    /scan



    Description:

    Scan your server and indentify problems! Get your protection score and fix the issues! To see how we calculate the protection score, check out the scan documentation.


    Permissions:

    Only the people with Administrator permissions can run this command.





    /inspect



    Values
    [ user: @username ] - The member you wanna inspect.



    Description:

    Get information about a specific user. The inspect command is the perfect command for just that. Get exclusive info about a member.


    Permissions:

    This command is available for everyone. Only the command runner can see the response.





    /ban-add



    Values
    [ user: @username ] - The member you wanna ban.
    [ reason: ] - Why are you banning this user?.
    [ proof: ] - Image of proof. ( OPTIONAL )



    Description:

    Ban a member from the server. User will be removed from the server, a log will be sent to moderation logs and also we'll notify the user as well.
    If you want, you can even add image proof with Meta!


    Permissions:

    Users with Ban Permissions can use this command. Also people with the Moderator Rank can run this command.





    /ban-remove



    Values
    [ user: @username ] - The member you wanna unban.
    [ reason: ] - Why are you unbanning this user.



    Description:

    Remove a ban from a user. A simple way to unban someone from the server. Simply list thier Discord ID and the reason and you're done!


    Permissions:

    Users with Ban Permissions can use this command. Also people with the Moderator Rank can run this command.





    /timeout-add



    Values
    [ user: @username ] - The member you wanna time out.
    [ timeout-format: ] - What is the format of your time. ( Seconds, Minutes, Hours, Days )
    [ time: ] - For how long you wanna mute the user.
    [ reason: ] - What is the reason for the timeout.
    [ proof: ] - Image of proof. ( OPTIONAL )



    Description:

    Set user to timeout. User who'se in timeout cannot join any voice channels and cannot chat in the server. Meta will take care of the logging, gathering proof and notifing the user.


    Permissions:

    Users with Moderate Member Permissions can use this command. Also people with the Moderator Rank can run this command.





    /timeout-remove



    Values
    [ user: @username ] - The member you wanna remove from timeout.
    [ reason: ] - Why are you removing the timeout.



    Description:

    A simple way to remove a timeout from a user. Meta will log the action and notify all parties.


    Permissions:

    Users with Moderate Member Permissions can use this command. Also people with the Moderator Rank can run this command.





    /ping



    Description:

    Bot being laggy? It might be because of some sort of host issue or something else. But the best way to know that is to check the ping.


    Permissions:

    This command is available for everyone. Only the command runner can see the response.
