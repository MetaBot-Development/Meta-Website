Meta Commands

Setup Command

/setup [log-channel: #choose-your-log-channel]


Set up your server Meta Log channel.


/setup [mod-rank: @your-main-moderator-rank]


Set the Mod Rank that Meta will monitor.


/setup [mute-rank: create]


Create the special Meta Ban role for protection systems.


/setup [protection: mod-grief | anti-alt | safe-bot | auto-mod]


Enable or Disable any protection systems.


Learn about each protection system here: • Anti Mod Grief • Anti Alt • Safe Bot • Auto Mod
