Dangerous Roles

What is a Dangerous Role?

Dangerous role is a made up term by the Meta Team to filter out more sensitive ranks that have the ability to harm your server.
If a role has a permission that is very strong and powerful, it is considered to be a dangerous role.

Users who have dangerous roles are not affected by Meta just because they have a dangerous role. However if they abuse their permissions and you have setted up protections for it, Meta does take action on them, no matter how powerful they are.

Dangerous roles are filtered out in your server the moment you add Meta, you don't have to do anything, we got this ;)



What is considered to be a Dangerous Role?

Each role is scanned by Meta, we're looking at the role permissions. If a role has at least one of the permissions (permissions listed below), it is considered to be a dangerous role.

Nobody can change / remove dangerous roles definitions. This is only modified by Meta Team.

Following permissions make a role Dangerous:

MANAGE_MESSAGES - User can delete and manage other user messages.

MANAGE_GUILD - User can change guild name, avatar and other general values.

MANAGE_ROLES - User can give out roles to other users. Can also edit, create and delete roles.

MENTION_EVERYONE - User can use the @everyone or @here mention tag that works.

MODERATE_MEMBERS - User can time out other users in the server.

BAN_MEMBERS - User can ban or unban other users from the server.

KICK_MEMBERS - User can kick other users from the server.

ADMINISTRATOR - User can do everything in the server, besides deleting the server or modifing the owner account values.
