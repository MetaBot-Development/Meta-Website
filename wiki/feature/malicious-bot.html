What is Malicious Bot Detection and how it works

The Malicious Bot Detection system protect your server from being griefed by malicious bots.
When this is enabled, Meta wont allow bots to be added into your server, unless the user who'se trying to add the bot has Administrator permissions.

When an user without Administrator permissions add a bot, the user will be Meta Muted and the bot will be just banned from the server.
A detailed log embed will be sent into Administration Logs channel.
