Auto Mod Protection System

Protect your server from the most common and needed things such as racism, server promotions, self botting and anti rule34.

What is Auto Mod and why we offer it

Auto Mod is short for Auto Moderation. Meta will take care of the most common Moderation tasks.
We offer 4 different Auto Mod rules and they are all enabled, once the Auto Mod module is enabled. You cannot enable or disable the rules individually.

Since Discord has started to provide their own Auto Mod, a lot of the systems are quite useless when it comes to auto moderation in bots. But there is a catch.
Meta makes you click 1 button, to enable 4 rules that all server should have enbled. Discord system is becoming more and more difficult and harder to understand as the time goes on, that's why we created our module.
If you want to make your server fancy and very protected and restricted, you can use Discord's automod, but Meta allows you to make the server safe and free, by clicking 1 button. Learn about the rules below.

No Racism

Meta has blacklisted over 400 ways to spead racism or say very disrespectful words. When an attempt is made, Meta will delete the message and warn the user in the DM's. Also, a detailed log will be sent into the Moderation Logs channel.

Discord Invites

Make it that people cannot send Discord server invites in your server channels. Meta will delete all links that contain discord.gg/ content, blocking and deleting all the discord invite links. When attempted, the message is deleted and the user is verbally warned via DM's. Also, a detailed log will be sent into the Moderation Logs channel.

Self Botting Meta Plus

Self botting is when user is behaving like a bot. They are sending embeds or doing other things that a general human can't do.

When such activity is detected, the user is instantly Meta Muted. An embed will be sent to the staff team and into the user DM's. The log channel will be the Moderation Logs channel.

No Rule34

Rule 34 is a "joke" that leads to NSFW content. To remove it from your server, enable Meta Auto Mod System!

Meta will delete all content that contains the phrase Rule34 in any type of form. The message is deleted, user is notified and a log message is sent into the Moderation Logs channel.
