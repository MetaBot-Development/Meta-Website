AutoMod

How does it work?

Meta does not have a custom AutoMod system anymore, instead, we work with Discord to provide you the best and most reliable experience when protecting your server with Meta. This system works even when Meta bot is offline / in maintenance.

Instead of Meta controlling everything itself, we have implemented Discords AutoMod into Meta, by setting up AutoMod with Meta, you'll be setting up your Discord Automod, but with Meta it's easier, cleaner and faster, like a lot faster.

Permission Warning

Users with the Administrator permissions will be ignored by Meta. Administrators can change the configurations of the module.

Action Logs

All AutoMod logs will be sent into the server mod-logs channel. You cannot change the logging channel for AutoMod.

How to Modify AutoMod?

In case you'd like to change how the AutoMod works, you'll have to head over to your server settings.

Casual servers: Server Settings -> AutoMod

Community servers: Server Settings -> Safety Setup -> AutoMod

After that, you can choose whatever rule you need to edit and do so! Example video below.
