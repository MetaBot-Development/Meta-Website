Anti Mod Grief

What is AMG

Anti Mod Grief (AMG) is a background protection system that prevents your server staff from harming your server. Limit your server staff from some more dangerous actions such as deleting a channel / role, unbanning manually without logging, giving out dangerous roles and much more!

AMG actions can be configured independently to give you as much freedom as possible.
To learn about action filters, check below.

Permission Warning

Users with the Administrator permissions will be ignored by Meta. Administrators can change the configurations of the module.

Action Logs

All Anti Mod Grief logs will be sent into the server admin-logs channel. You cannot change the logging channel for Anti Mod Grief.

Actions

Users who have the Moderator Rank will be affected by this, users without it wont be affected.

Channel Created

User cannot create any type of channels

Channel Deleted

User cannot delete any channels

Channel Modified

User cannot modify channel permissions / fields

Manual Ban Remove

Users must use an app to ban other members

Dangerous Role

User cannot give / remove Dangerous Roles to / from other members / bots

Guild Update

User cannot change server overview settings

Role Created

User cannot create any roles

Role Deleted

User cannot delete any roles

Role Updated

User cannot modify / update any role

What each action has:

Action logs will be sent into admin-logs

Only takes action on users who have the Moderator Rank

Ignores Administrators and Owners

Permissions Needed

Make sure Meta has all the necessary permissions, if you're not sure, try giving the app Administrator permissions.
