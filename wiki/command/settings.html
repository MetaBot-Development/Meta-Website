In total, Meta has 11 public commands that server administrators, managers and other staff can user.
Meta's commands are separated into 3 different categories, moderation, information and server.

/settings

Settings command is the main command when using Meta Bot, as this command shows your server settings.
Via the settings command, you can also modify the protection modules Anti Mod Grief, Anti Alt, Malicious Bot Detection and Auto Mod.
