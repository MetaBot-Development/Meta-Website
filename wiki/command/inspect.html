INFORMATION

In total, Meta has 11 public commands that server administrators, managers and other staff can user.
Meta's commands are separated into 3 different categories, moderation, information and server.

/inspect [user]

Inspect is a simple and quick way to find general information about an user like account dates, profile customization, Meta Status and server information.

User

The person / member / user whose profile will be show. This user must be in the same server where you are running the command.
